# KTTTOS
## 项目介绍
本项目是由KTTT所主导开发、隶属于KTTT Studio的操作系统项目，启动于2024.1.12

构建在UEFI(edk2)环境上，使用Ubuntu20.04LTS系统编译。

## 环境依赖
编译用操作系统：Ubuntu20.04LTS

Kernel编译器：GCC5.0

BootLoader编译器：edk2

虚拟机测试：Qemu
